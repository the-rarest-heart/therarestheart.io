VM Setup Guide

Get Your Lab Environment Running

You need a Windows 10 or 11 VM to practice every procedure in this curriculum. Free options below — no license purchase required for evaluation builds.

💡

Recommended setup: Windows 10 22H2 evaluation ISO in VirtualBox on any host OS. Allocate 2 CPU cores, 4 GB RAM minimum. This matches the environment referenced in all module SOPs.

Step 01 — Hypervisor

Install VirtualBox

VirtualBox is free, cross-platform, and works on Windows, macOS, and Linux hosts. Download the latest stable release and install Guest Additions for clipboard sharing.

virtualbox.org/wiki/Downloads →
Step 02 — OS Image

Download Windows Evaluation ISO

Microsoft offers 90-day evaluation builds free of charge. Download the Windows 10 Enterprise evaluation ISO from the Microsoft Evaluation Center — no product key needed.

Evaluation Center →
Step 03 — Create VM

Configure the Virtual Machine

New → Windows 10 (64-bit). Assign 4096 MB RAM and 2 CPUs. Create a dynamic VDI disk with at least 60 GB. Mount the ISO, boot, and run through Windows setup.

Step 04 — Snapshot

Take a Clean Snapshot

After Windows setup completes, take a VirtualBox snapshot named Clean Install. This lets you revert to a fresh state between labs without reinstalling the OS.

Step 05 — Tools

Install Lab Tools

Download Malwarebytes Free, a cable tester app sim, and RSAT (for AD labs). All are referenced in module SOPs and installable in the evaluation VM.

Step 06 — Practice

Start Lab 01

Open the Password Resets module, switch to Phase 2 (Do IT), and run through SOP-001 step by step. Every SOP is designed to be followed in your lab VM.
