Common Questions

Frequently Asked

Quick answers to questions I get asked regularly.

Is this curriculum free?

Yes — everything on this site is free. The modules, SOPs, and practice scenarios are openly available to anyone working toward an IT career.

What experience level is this for?

Tier 1 modules (01–05) are designed for complete beginners. Tier 2 modules (06–09) assume basic help desk familiarity. No degree or prior experience required.

Can I use this content to study for CompTIA A+?

Absolutely. The topics covered — hardware, networking, OS troubleshooting, security, and Active Directory — map closely to A+ Core 1 and Core 2 exam objectives.

Are there hands-on labs?

Lab environments are in progress. Each module's "Do IT" phase outlines the exact procedure — you can follow along in a Windows VM or TryHackMe room while they're being built out.

How often is new content added?

New modules, labs, and scenario content are added as the curriculum is built out publicly. Follow on GitHub or check back — the site evolves alongside the learning journey.

Can I contribute or suggest a topic?

Yes — open an issue on GitHub or send a message via the contact form. Topic suggestions, corrections, and pull requests are all welcome.
